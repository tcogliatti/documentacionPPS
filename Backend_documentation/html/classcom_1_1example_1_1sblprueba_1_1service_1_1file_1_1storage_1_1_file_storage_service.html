Descripción detallada

FileStorageService

Clase abstracta que contiene todos los metodos para hacer ABM con archivos. La dirección en la cual se realizaran esos ABM se define en cada instancia concreta que extienda de esta. Ademas, las instancias concretas de esta clase deberá implementar el método validateFile() de acuerdo a los criterios particulares de validación aplicables a cada caso
